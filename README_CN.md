# Quick Save (保存历史工具)

一个高效的 Photoshop UXP 插件，专注于快速保存和覆盖文档。它能记录您的“另存为”历史，并允许您通过简单的双击操作将当前文档覆盖保存到历史路径中。

## ✨ 主要功能

*   **💾 另存为并记录**：通过插件的“另存为”按钮保存文档，自动记录文件路径和缩略图。
*   **⚡ 快速覆盖**：**双击**历史列表中的缩略图，即可将当前打开的文档覆盖保存到该位置。
*   **🖼️ 高性能缩略图**：采用本地缓存机制生成 128x128 高清缩略图，加载迅速且不占用文档体积。
*   **🌍 多语言支持**：界面自动适应 Photoshop 语言设置（支持简体中文和英文）。
*   **🔒 权限管理**：智能管理文件写入权限，通过视觉指示器（左上角绿点）显示权限状态。
*   **🎨 紧凑界面**：专为专业工作流设计的紧凑 UI，支持深色模式。

## 🚀 安装方法

### 方法 1：双击安装 (推荐)
1.  获取 `.ccx` 安装包。
2.  双击文件，按照 Adobe Creative Cloud 提示完成安装。

### 方法 2：开发者模式安装
1.  安装 **Adobe UXP Developer Tool**。
2.  在 Photoshop 首选项中开启“启用开发者模式”。
3.  在 UXP Tool 中点击 "Add Plugin"，选择本插件目录下的 `manifest.json`。
4.  点击 "Load"。

## 📖 使用指南

### 1. 添加记录
点击插件面板顶部的 **"另存为" (Save As)** 按钮。
*   选择保存格式（PNG, JPG, TGA, PSD）。
*   选择保存位置。
*   保存成功后，该文件会自动添加到下方的历史列表中。

### 2. 快速覆盖
当您需要将当前文档更新到某个历史版本时：
*   在列表中找到目标文件。
*   **双击**该缩略图。
*   确认提示框后，当前文档将直接覆盖目标文件。

### 3. 管理历史
*   **查看路径**：鼠标悬停在缩略图上可查看完整文件路径。
*   **删除记录**：点击缩略图右上角的 **"×"** 按钮删除单条记录（同时删除对应的缩略图缓存）。
*   **清空历史**：点击底部的 **"清空" (Clear)** 按钮删除所有记录和缓存。

## ⚙️ 系统要求

*   Adobe Photoshop 2021 (v22.1.0) 或更高版本。

## 📝 注意事项

*   **覆盖操作不可撤销**：双击覆盖保存会直接替换原文件，请谨慎操作。
*   **权限提示**：如果缩略图左上角显示绿色小点，表示插件当前拥有该文件的写入权限；如果没有，覆盖时可能会再次请求权限。
*   **缓存清理**：删除历史记录时会自动清理对应的本地缩略图缓存文件，保持系统整洁。
